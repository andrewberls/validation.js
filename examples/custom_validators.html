validation.js Example - Custom Validators

This is an example of using custom user-defined validator functions. The sample validator here enforces that only the number 2 is a valid input (for very important reasons), and is defined using a name and a validation function that returns true for valid inputs and false for invalid ones. 

ValidateJS.addValidator('isTwo', function(input) {
  return parseInt(input) === 2;
});

You can declare an input to use a custom validator using the data-validator attribute:

<input type="text" name="magic_number" data-validator="isTwo">
